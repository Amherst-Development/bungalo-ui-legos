# Welcome to Bungalo UI

## Getting Started
**Run Docs:** `yarn docz:dev`

**Build Components:** `yarn lib`

**Run Tests:** `yarn test`

## Test in bh-web
- Create Branch in bungalo-ui-legos
- Do your work on a component
- run `yarn lib` to build the component library
- commit & push your changes and the lib build
- Go to bh-web repo
- in `package.json` bungalo-ui-legos dependency change the info after the `#` to your branch name
- ex: `"bungalo-ui-legos": "github:bungalohomes/bungalo-ui-legos#v0.1.0"` to `"bungalo-ui-legos": "github:bungalohomes/bungalo-ui-legos#your-branch-name"`
- in bh-web repo run `rm -rf node_modules/bungalo-ui-legos && npm install` to get your latest code.
- Test away!

## Deploy your new UI changes
- Create Branch in bungalo-ui-legos
- Do your work on a component
- Go to the main `package.json` and increment the version number. 
- run `yarn lib` to build the component library
- commit & push your changes and the lib build
- create PR & merge to master. 
- **in git** tag your merge with the incremented version number in this format: `vX.X.X (v0.1.0)`
- Push your tag that you created.
- Go to bh-web repo
- in `package.json` bungalo-ui-legos dependency change the info after the `#` to your new version
- ex: `"bungalo-ui-legos": "github:bungalohomes/bungalo-ui-legos#v0.1.0"` to `"bungalo-ui-legos": "github:bungalohomes/bungalo-ui-legos#v0.1.1"`
- in bh-web repo run `rm -rf node_modules/bungalo-ui-legos && npm install` to get your latest code.
- Now you can use your new or changed components!
