# Colors
Colors are found and edited inside `lib/colors/colors.scss` and `lib/colors/colors.js`

When you add a new color, be sure to add it to both the scss and js files. 

Look at `lib/components/Button/Button.jsx` to see how to import and use the colors in these components.
