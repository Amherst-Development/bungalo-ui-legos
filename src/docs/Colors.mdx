# Colors
Colors are found and edited inside `lib/colors/colors.scss`

When you add a new color, be sure to export it inside of that scss file. 

Look at `lib/components/Button/Button.jsx` or `lib/colors/ColorSwatch.jsx` to see how to import and use the colors in these components.

Eventually we will export this scss file to be used as the color template across our application.
